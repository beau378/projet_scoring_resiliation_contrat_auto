🔍 Projet de Scoring : Résiliation des Contrats Auto

Ce projet vise à modéliser le risque de résiliation des contrats d’assurance automobile à partir d’un jeu de données de 90 000 clients. Il met en œuvre des techniques avancées de data science : feature engineering, modèles supervisés (XGBoost, LightGBM, stacking), analyse SHAP, et visualisation dynamique sous Power BI.

📊 Voir le Dashboard 💻 Code sur GitHub

🎯 Objectif

Prédire quels clients risquent de résilier leur contrat auto afin d’aider à la fidélisation et à la prise de décision marketing.

🛠️ Méthodologie

  • Nettoyage des données et encodage des variables catégorielles
  • Création de variables agrégées (contrats actifs/résilés par branche)
  • Entraînement de plusieurs modèles : logistique, XGBoost, LightGBM, stacking
  • Évaluation par F1-score, AUC, lift, matrice de confusion
  • Analyse SHAP pour interprétabilité
  • Visualisation dynamique via Power BI

📂 Contenus du projet

  • Notebooks Google Colab : prétraitement, modélisation, visualisation
  • Dashboard Power BI (SharePoint)
  • Explication détaillée dans le README du dépôt GitHub
