Scoring dynamique de résiliation de contrats auto

Ce projet explore l'utilisation de modèles prédictifs et de Power BI pour identifier les clients à risque de résiliation.

Méthodologie Dashboard Analyse Variables clés GitHub

🧠 Méthodologie

Le projet suit un pipeline classique de machine learning :

  • 📊 Exploration : compréhension des données, détection des outliers, visualisation des distributions.
  • 🛠 Préparation : imputations, encodages (One-Hot, Label Encoding), normalisations.
  • 📈 Modélisation : modèles testés : Régression logistique, Random Forest, XGBoost. Optimisation par recherche bayésienne.
  • 📌 Seuil décisionnel : ajusté selon les objectifs business, courbe de lift pour affiner la stratégie de ciblage.

📌 Analyse des résultats

  • 📉 Taux moyen de résiliation : 8,4%
  • 🟠 Risque élevé : représente 7% des clients, avec un taux de résiliation de 38%
  • 🟢 Risque faible : 93% des clients, taux de résiliation de 6%
  • 📈 Lift : performance 4 fois supérieure au hasard pour les clients les plus risqués
  • 🧩 Matrice de confusion : 20206 vrais négatifs, 588 vrais positifs

🔍 Variables clés de profil à risque

Les profils à risque sont caractérisés par :

  • Marques de véhicule : Citroën, Peugeot, Renault
  • Formule souscrite : C, S, T
  • Montant payé : entre 316 et 29 473 €
  • Ancienneté client : entre 11 688 et 14 664 jours
  • Niveau bonus-malus : 50, 95, 100
  • Type d’agent : 611, 613, 601
  • Fréquence sinistre : variables DI, RESMH, RESDI, MH, SA
