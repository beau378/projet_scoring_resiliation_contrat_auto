📊 Projet Data : Prédiction de la résiliation des contrats auto

Ce projet vise à prédire la résiliation de contrats d'assurance automobile à partir des données de 90 000 clients. Il mobilise des techniques avancées de data science, de la modélisation supervisée à l’interprétation (SHAP), et se conclut par un dashboard interactif sous Power BI.

🔎 Voir le dashboard interactif 💻 Code GitHub

🧠 Compétences mises en œuvre

  • Feature engineering sur données clients (contrats actifs/résilés)
  • Modélisation : régression logistique, XGBoost, LightGBM, stacking
  • Évaluation : AUC, F1-score, courbe de lift, matrice de confusion
  • Analyse des variables importantes (SHAP)
  • Visualisation métier sous Power BI
