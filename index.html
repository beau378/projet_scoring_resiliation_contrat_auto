Scoring de Résiliation Auto

Analyse prédictive du risque de résiliation à partir des données clients.

Méthodologie & Traitement

Le projet commence par une phase d’exploration des données. Les variables sont ensuite préparées à l’aide d’encodages et d’imputations adaptés.

Deux modèles sont testés : une régression logistique et un XGBoost. Leur performance est optimisée par une recherche bayésienne.

Un seuil de décision est enfin défini, afin de cibler efficacement les clients à risque et maximiser le score global du modèle de prédiction.

Dashboard & Analyse

1. Résultats globaux

  • Nombre de clients : 23 095 individus sont inclus dans le jeu de test
  • Score moyen de résiliation : 7,9 % — moyenne des probabilités prédites
  • Taux moyen de résiliation : 8,4 %
  • Risque élevé : représente 7 % des clients, avec un taux de résiliation de 38 %
  • Risque faible : 93 % des clients, taux de résiliation de 6 %
  • Lift : performance 4 fois supérieure au hasard pour les clients les plus risqués
  • Matrice de confusion : 20 206 vrais négatifs, 588 vrais positifs

2. Profil des clients à risque

  • Marques de véhicule : Citroën, Peugeot, Renault
  • Formule souscrite : C, S, T
  • Montant payé : entre 316 et 29 473 €
  • Ancienneté client : entre 11 688 et 14 664 jours
  • Niveau bonus-malus : 50, 95, 100
  • Type d’agent : 611, 613, 601
  • Fréquence sinistre : variables DI, RESMH, RESDI, MH, SA
